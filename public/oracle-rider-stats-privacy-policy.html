Privacy Policy for Oracle Rider Stats (Chrome Extension)

Effective date: March 9, 2026

1. Who we are

This privacy policy applies to the Chrome extension Oracle Rider Stats published by Oracle Games.

2. Data processed by the extension

The extension processes only data needed to show rider statistics:

  • Rider name and/or rider slug from Oracle Games when a user clicks "PCS stats".
  • Public rider profile data fetched from ProCyclingStats.

The extension does not collect passwords, payment details, or sensitive personal user data.

3. Permissions and network access

  • The extension uses host permission https://www.procyclingstats.com/* to fetch public rider profile information.
  • The extension does not load or execute remote code (no remote JavaScript or CSS execution).

4. Purpose of processing

Data is processed only to display rider statistics inside Oracle Games for rider comparison and team selection support.

5. Data sharing and selling

We do not sell personal data. We do not share data with third parties except requests sent to ProCyclingStats to fetch public rider information requested by the user.

6. Data retention

The extension does not maintain a server-side user database for this feature and does not intentionally retain personal data from extension usage.

7. Contact

For privacy questions, contact: no-reply@send.oracle-games.online

8. Changes to this policy

We may update this policy when product behavior or legal requirements change. The latest version will always be available at this exact URL.
